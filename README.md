# AI Placement Mentor

AI Placement Mentor is an intelligent career companion designed to help students prepare for placements through personalized resume analysis, mock interviews, career guidance, and long-term memory.

Instead of treating every interaction independently, the platform connects resume insights, interview performance, and career goals to provide a continuous and personalized placement preparation experience.

## Features

### AI Chat
Ask career-related questions, placement queries, learning roadmaps, and technical concepts through an AI-powered assistant.

### Resume Analyzer
- ATS Score Analysis
- Resume Strengths & Weaknesses
- Skill Gap Detection
- Project Recommendations
- Interview Question Suggestions
- Personalized Learning Roadmap

### Mock Interview
- Resume-based Interview Questions
- Personalized Question Generation
- Performance Evaluation
- Detailed Feedback & Improvement Areas

### Career Copilot
- Career Path Recommendations
- Fresher Salary Insights
- Target Company Suggestions
- Skill Development Guidance

### Persistent Memory
Powered by Hindsight, allowing the system to remember:
- Career Goals
- Resume Feedback
- Interview Performance
- Learning Progress
- User Preferences

### Intelligent Routing
Powered by CascadeFlow to efficiently route requests between different AI workflows and modules.

## How It Works

1. Upload your resume
2. Receive ATS analysis and feedback
3. Practice resume-based mock interviews
4. Get interview evaluation reports
5. Receive personalized career recommendations
6. Continue improving with long-term memory support
